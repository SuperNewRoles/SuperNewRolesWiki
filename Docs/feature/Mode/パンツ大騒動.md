# ぱんつ！
- 生き残りをかけて、パンツを奪い、パンツを守るモード！
# 詳細
- ゲームを開始すると全員の中から半分のプレイヤーにパンツが配られます。
- 始まると、ランダムなスポーン場所にテレポートします。
- ぱんつを持ってない人はぱんつを持ってる人からパンツを奪おう!
- パンツを奪われたら、奪えるまではクールタイムがあるぞ!このクールタイムはバニラ設定のキルクールタイムの半分です。
- そして、制限時間までにパンツを履いていなかったら死亡し、またパンツが配られます。
- そこからまたテレポートし、残りの一人が決まるまで繰り返します！
- 最後に生き残ってた君がぱんつの戦いの勝者だ!
# 設定箇所
- キルクールタイム
   - ぱんつを奪われた際に、次回に奪えるようになるまでの時間です。キルクールタイムの半分が使用されるので、設定したいクールタイムの2倍の時間を設定しましょう。
# ストーリー(ChatGPT原案、一部変更あり)
宇宙船の乗組員たちは、長期間の任務を完了するため、膨大な数のパンツを準備しました。しかし、ある日、彼らはパンツが不足していることに気づきました。そのため、パンツを持っている乗組員たちは貴重な資源を手に入れたと感じ、自分たちだけがパンツを所有することを望み始めました。

最初のうちは、乗組員たちは相手にパンツを譲ることもありましたが、次第に競争心が高まり、パンツを奪い合うようになりました。パンツを持っている者たちは、常に注意深く、相手にパンツを奪われないように、周囲を警戒しました。

パンツが足りないという事態は、次第にエスカレートし、船内は混乱の渦に巻き込まれました。乗組員たちは、どんどん暴力的になり、船内で激しい戦いが繰り広げられるようになりました。制限時間が終了するまでにパンツを持っていなかった乗組員は、不衛生により消息を絶ち、最後に残った乗組員がパンツを手にすることができました...
