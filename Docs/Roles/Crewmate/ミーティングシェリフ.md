## 役職説明
- アクティブ能力：**ミーティングキル**
  - 会議中に対象の名前の横に表示されるボタンを押すことで対象に能力を執行することが出来る。

| 名前 | 詳細 |
| :-- | :-- |

| 執行成功時及び誤射時の挙動設定 | 能力を行使した際の挙動の設定 |
