- SuperHostRoles対応済み

## 役職説明
- アクティブ能力：**シェイプ**
  - バニラ役職のシェイプシフターと同じような変身をする。
  - 変身した相手をキルするとキルクールが短くなる。
  - それ以外の相手をキルしたり、変身せずキルをするとキルクールが長くなる。

## ゲーム設定
| 名前 | 詳細 | SHR対応 |
| :-- | :-- | :--: |
| 持続時間 | 変身していられる時間(秒) | ◯ |
| クールタイム | 変身出来るようになるまでの時間(秒) | ✕ |
| 成功時のキルクール | 変身した対象をキルしたときのキルクール(秒) | ◯ |
| 失敗時のキルクール | 上記以外の相手をキルしたときのキルクール(秒) | ◯ |
