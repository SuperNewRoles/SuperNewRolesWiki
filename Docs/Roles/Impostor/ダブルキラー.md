## 役職説明
- アクティブ能力：**ダブルキル！**
  - この役職は2つキルボタンを持つ。
  - 2つのキルボタンそれぞれが別のクールタイムを持つ。
  - その代わり、サボタージュやベントを使用出来ない(設定)。

## ゲーム設定
| 名前 | 詳細 |
| :-- | :-- |
| メインキルボタンのクールダウン | 1つめのキルボタンのキルクールタイム(秒) |
| サブのキルボタンのクールダウン | 2つめのキルボタンのキルクールタイム(秒) |
| サボを使えるか | サボタージュを使用出来るか |
| ベントが使える | ベントを使用出来るか |
