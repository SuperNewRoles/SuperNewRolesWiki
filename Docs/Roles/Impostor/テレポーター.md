## Description

## 役職説明
- アクティブ能力：**テレポート！**
  - ランダムに選ばれた対象の位置に、生存者を全員テレポート(集合)させる。
  - 能力使用後即テレポートはされず、[テレポートされるまでの時間]で設定した秒数が経過してから、テレポートする。

## ゲーム設定
| 名前 | 詳細 |
| :-- | :-- |
| テレポート能力のクールタイム | [テレポート!]能力の使用に必要な時間[s]を設定します。 |
| テレポートされるまでの時間 | テレポーターが能力を発動してからテレポートされるまでの猶予時間を設定します。 |
