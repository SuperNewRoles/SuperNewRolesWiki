## 役職説明
- アクティブ能力：**必殺技**
  - 1試合に1回だけ使用出来る。
  - 使用すると自身の周囲にいるクルー全てをキルする。
  - その代わり、この役職はベントやサボタージュを使用できない(設定)。

## ゲーム設定
| 名前 | 詳細 | SHR対応 |
| :-- | :-- | :--: |
| キルクールダウン | この役職固有のキルクール(秒) | ◯ |
| 必殺技のクールダウン | 「必殺技」のクールタイム(秒) | ✕ |
| ベントが使える | ベントが使用出来るか | ◯ |
| サボタージュが使える | サボタージュを使用出来るか | ◯ |
| 必殺技が及ぶ範囲 | 「必殺技」の効果範囲 | ◯ |
