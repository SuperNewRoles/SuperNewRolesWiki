SuperHostRoles対応

  • SuperHostRoles対応済み

イントロ

  • 皆の気配を感じ取ろう

Description

  • 死亡時刻や死亡場所を把握する事が可能な、死者に関わる情報を司る役職です。
  • 相方の行動を把握しサポートに回りましょう。
  • 生存者に死霊を憑ける呪いをかける設定や、自身が死亡した時に狼憑きとして取り憑く設定もあります。
  • 貴方の周りには仲間がいる。さみしくはない。
  • いずれ貴方もその一員に入るのだ。

役職説明

  • 基本能力としてシーアと同じ能力を有しますが, シーアよりも能力が強化されています。

  • パッシブ 1：死の点滅が見える

    • キルが発生した際、画面が光る。

    • 設定 : [死の点滅をクルーのボディカラー依存にするか]
      • 有効時 :
        • [死者ボディカラーの表示方法] : 色の詳細がわかる
          • 死の点滅の色が, 死者のボディカラーになります。

        • [死者ボディカラーの表示方法] : 明暗のみがわかる
          • 死者の色が明るい時 => 死の点滅の色が, 勿忘草色になります。
          • 死者の色が暗い時 => 死の点滅の色が, 暗紅色になります。

      • 無効時 : 死の点滅の色が, 青色になります。

    • 設定 : [ボディカラーの情報を文字でも表示するか]
      • 死の点滅の補助機能です。
      • 有効時 :
        • [死者ボディカラーの表示方法] : 色の詳細がわかる
          • 死者のボディカラーの名前を文字で通知します。
            • 通知内容 : 「クルー名 ■ のクルーの気配が弱まりました。」
            • ■ は死者のメインカラーで表示されます。

        • [死者ボディカラーの表示方法] : 明暗のみがわかる
          • 死者の色が明るい時 => 「明るい色のクルーの気配が弱まりました。」と通知されます。
          • 死者の色が暗い時 => 「暗い色のクルーの気配が弱まりました。」と通知されます。

  • パッシブ 2：霊魂が見える(SHR非対応)
    • 誰かが死亡した会議の終了後、死亡位置に霊魂が現れる。
    • 設定「時間経過で霊魂が消える」の挙動
      • オフにしている場合、再度会議を挟んでもそこに霊魂が表示された状態になる。
      • オンの場合、設定時間が会議終了から設定時間が経過すると霊魂が見えなくなる。

    • 設定 : [霊魂をクルーのボディカラー依存にするか]
      • 有効時 :
        • [死者ボディカラーの表示方法] : 色の詳細がわかる
          • 霊魂の色が, 死者のボディカラーになります。

        • [死者ボディカラーの表示方法] : 明暗のみがわかる
          • 死者の色が明るい時 => 霊魂の色が, Modカラー[真っ白]になります。
          • 死者の色が暗い時 => 霊魂の色が, Modカラー[クラッシュブルー]になります。

      • 無効時 : 霊魂の色が, Modカラー[クラッシュブルー]になります。

  • パッシブ 3: 死体位置矢印(SHR非対応)
    • 死体が存在する場合, 死体の方向に矢印が表示されます。

    • 設定 : [死体位置矢印をクルーのボディカラー依存にするか]
      • 有効時 :
        • [死者ボディカラーの表示方法] : 色の詳細がわかる
          • 矢印の色が, 死者のボディカラーになります。

        • [死者ボディカラーの表示方法] : 明暗のみがわかる
          • 死者の色が明るい時 => 矢印の色が, Modカラー[真っ白]になります。
          • 死者の色が暗い時 => 矢印の色が, Modカラー[クラッシュブルー]になります。

      • 無効時 : 矢印の色が, Modカラー[クラッシュブルー]になります。

  • アビリティ 4:取り憑かせる
    • ターゲットに重複役 : 狼憑きを付与します。
    • 作成した狼憑きの設定は, 狼憑きのロール設定に依存します。

  • パッシブ 5: 取り憑く (SHR非対応)
    • 自身がキルされた時に, 自身をキルしたプレイヤーに狼憑きを付与します。

  • アビリティ 6: 狂わせる (SHR非対応)
    • 対象を役職問わずにマッドメイトにする事が可能です。
    • 狼憑きを作成する能力とは共存しません。
