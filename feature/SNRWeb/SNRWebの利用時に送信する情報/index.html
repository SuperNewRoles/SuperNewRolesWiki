SNRWebの利用時に送信する情報

SNRWebへの情報送信

送信される条件

🔵info :
条件が満たされている時, welcomeメッセージにその旨の情報が表示されます。


  • ホストがSNR上でWebサービスにログインしている。
  • ホストがSNR上でWebサービスにログインしていないが, [ホストがWeb未ログイン時，ゲストからの戦績の送信を許可する]設定が有効で, SNR webにログインした導入者ゲストが参加している。

表示される情報 詳細

条件表示される情報
ホストがログイン状態の時
ホストが未ログインで,
[ホストがWeb未ログイン時，ゲストからの戦績の送信を許可する]設定が有効な時
(ログイン状態の導入者が参加していない場合も送信される為,
この情報表示時に必ず送信されるとは限らない。)

送信される内容

⚠Warning :
以下の情報は送信される条件が満たされている場合, 登録及びログイン状態に関わらず参加者全員の内容を送信します。


  • フレンドコード

  • 試合結果
  • ゲーム情報
    • 使用モード
    • 使用マップ
    • 参加人数
    • 会議回数

  • リザルト
    • 勝利陣営

  • 会議情報
    • 会議開始時 生存人数
    • 会議開始時 死亡人数
    • 同ターンのタスクフェイズ中に死亡した人数
    • 追放者

  • プレイヤー情報
    • プレイヤー名
    • プレイヤーカラー
    • プレイヤーの役職
    • 完了タスク数

送信される情報の利用方法

  • SNRWebに送信した情報の利用方法のページを確認して下さい。
