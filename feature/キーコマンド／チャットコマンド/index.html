キー説明
F3参加者のプレイヤー情報を表示します。
フレンドコードは [バニラ設定 : 配信者モード] 有効時**********#****と伏字として表示されます。

ロビー及び会議ターン

キー説明
F1チャットを開閉します。
F2チャットを再起動します。
(AmongUs v2022.3.29)であったチャットが開けなくなるバグの対処に使用されていました。
LeftControl + Cチャットに打ち込んだ文字をクリップボードにコピーします。
windows標準搭載のクリップボードを使用可能にしています。

Windows + VチャットでWindowsの標準機能であるクリップボードの履歴機能を使用する事ができます。
LeftControl + Xチャットに打ち込んだ文字をクリップボードに切り取ります。

チャットコマンド

全員

コマンド引数説明
/welcome
wlc		WelcomeMessageを再送します。(ホストが実行すると全員に送信します)
/version
/v		バージョンを表示します。
/commands
/cmd		コマンド一覧を表示します。
/GetWebInfo
/gwi		SNRwikiに関する簡易説明を送信します。(ホストでも自分のみに送信します)
詳細説明については[[SNR Webについて]]のページから確認して下さい。
/discord
/dc		公式Discordのリンクを表示します。
/twitter
/tw		公式Twitterと開発状況Twitterのリンクを表示します。
/GetInRoles
/gr		入っている全ての役職を表示します。
ホストが使用した場合、全員に送信されます。
/GetInRoles myplayer
/gr mp
/gr myp		入っている全ての役職を自分のみに送信します(ホスト向け)。
/AllRoles
/ar		送信間隔(半角数字)
未入力でも使用可能		記入例：/ar 0
入っている全ての役職の説明と設定が確認出来ます。
ホストが使用した場合、全員に送信されます。
/AllRoles myplayer
/ar mp
/ar myp		送信間隔(半角数字)
未入力でも使用可能		記入例：/ar 0 mp
入っている全ての役職の説明と設定を自分のみに送信します(ホスト向け)。
/roleinfo
/ri		役職名(*1)役職名から役職の情報を取得できます。ホスト以外は現在配役されている役職のみ検索ができます。
*1 ホストの設定言語依存. ホストが設定している言語が日本語ならば, 日本語の役職名で指定可能です。
又, 内部的な役職名(RoleId名)も使用可能です。ホストが日本語でもMadJester等, 検索する事で取得できます。
/BattleRoles
/btr		バトルロイヤルモードの全役職を確認する事ができます。
/Winners
/w		前回試合のリザルトをチャットで送信します。(ホストが実行すると全員に送信します)
/MatchTag
/tag		現在設定されているマッチメイキングタグを表示します。
/SaveLog
/sl		ファイル名(全角可)
未入力でも使用可能		記入例：/sl 突然全員落ちた試合のログ
その時点までのログを出力します。
ログはSuperNewRoles\SaveLogFolderに保存されます。
バグ報告時に提出頂けると幸いです。
